# AI-XR-Project

#  Mech-Raid XR

공간 컴퓨팅 기반 다중 사용자 협력 피트니스 보스 레이드 프로젝트입니다. 기존 VR/XR 환경의 HMD 착용으로 인한 멀미와 물리적 단절감을 극복하기 위해, 실제 공간에 디지털 요소를 투사하고 LiDAR 센서로 움직임을 정밀 추적하는 차세대 XR 플랫폼을 지향합니다.

##  프로젝트 개요

* **개발 목적:** MMRRPG의 핵심 요소인 ‘보스 레이드’ 기믹을 오프라인 다 인원 XR 환경에 융합하여, 스포츠와 게임의 경계를 허무는 협동형 액션 콘텐츠를 제시합니다.
* **기대 효과:** 몬스터와 광역 레이저를 피하기 위해 끊임없이 움직이며 자연스럽게 심폐지구력을 향상시키고, 기믹을 파훼하는 과정에서 팀워크와 공간 지각 능력을 극대화할 수 있습니다.
* **개발 기간:** 2026.6.29 ~ 2026.8.27 (총 9주).

##  팀 구성 및 역할

* **팀장 박건령:** 프로젝트 총괄, 서버 아키텍처 구축 및 게임 진행 제어, LiDAR 2D/3D 클라우드 좌표 데이터 로직 처리 및 공간 알고리즘 구현.
* **팀원 김시현:** 클라이언트 구현, 언리얼 엔진 기반 CAVE 및 nDisplay 프로젝션 환경 세팅, Meta Quest 3 프로토타이핑, 에셋 배치 및 시각 이펙트 최적화.
